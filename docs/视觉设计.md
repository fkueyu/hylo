# Hylo “Apple Native (2026)” 设计架构深化方案

随着用户对应用沉浸感与原生体验的要求不断提高，我们将 Hylo 的视觉语言全面转向 **Apple Native (macOS/iOS 最前沿的系统级设计语言)**。这并非简单地将颜色改为蓝色，而是从“色彩映射”、“材质表现”、“交互微动效”三个维度，打造如同苹果第一方应用（如 Xcode、SwiftUI 预览器）般的高级质感。

---

## 🎨 一、核心色彩体系 (Color System)

告别泛滥的“AI 靛紫色”，全面拥抱 Apple 极其严谨的 Human Interface Guidelines 动态系统色。

### 1. 品牌交互主色 (System Blue)
- **Light Mode**: `#007AFF` —— 经典、明亮，用于轻量环境下的视觉引导。
- **Dark Mode**: `#0A84FF` —— 亮度略高，确保在深色背景下保持足够的文本对比度与纯净的视觉穿透力。
- **用途**: 文件名脏值星号 (`*`)、选中文本高亮、右键菜单的 Hover 激活状态、拖拽条 (Splitter) 激活状态。

### 2. 空间与层级背景色 (Z-Axis Backgrounds)
苹果的原生质感源自于“对空间深度的精准切片”。
- **深色模式底层 (Base)**: `#1C1C1E` (System Gray 6) —— 取代死板的纯黑或 VSCode 的 `#1E1E1E`，带有一丝微妙的深空灰。
- **深色模式浮层 (Elevated)**: `#2C2C2E` (System Gray 5) —— 用于右键菜单、悬浮提示框 (Tooltip)。
- **浅色模式底层 (Base)**: `#F2F2F7` (系统标准窗口背景色)。
- **浅色模式面板 (Panel)**: `#FFFFFF` (纯白代码区)，形成干净的内容聚焦区。

### 3. 文字与次要信息 (Typography & Secondary)
- **主要文本**: 纯白 (Dark) / 纯黑 (Light)
- **次要文本 (统计信息/提示)**: `#8E8E93` (System Gray) —— 苹果最经典的辅助文本色，提供刚好足够的阅读清晰度而不抢戏。

---

## 🧊 二、材质与物理隐喻 (Materials)

2026 时代的苹果设计，去除了多余的阴影与边框，极度依赖材质(Materials)来区分层级：
1. **毛玻璃特效 (Vibrancy & Glassmorphism)**：
   - 全局右键菜单 (ContextMenu) 将采用 `backdrop-filter: blur(24px) saturate(180%)`，并配合 `rgba` 透明背景，在重叠代码时呈现出绝美的模糊透视感。
2. **极细的亚克力边框 (Hairline Borders)**：
   - 摒弃粗糙的 `#333` 边框。
   - 深色模式下，菜单与面板的边框采用 `border: 1px solid rgba(255, 255, 255, 0.1)`。
3. **丝滑阴影 (Smooth Shadows)**：
   - 悬浮元素的阴影从生硬的单层扩散改为多层拟真阴影，例如 `box-shadow: 0 8px 32px rgba(0,0,0,0.24), 0 2px 8px rgba(0,0,0,0.12)`。

## ⚡ 三、具体 CSS Tokens 重构清单 (即将实施)

接下来，我将向 `src/index.css` 注入以下原生的设计代币（Design Tokens）：

```css
/* 基础与原生蓝色系 */
:root {
  --color-primary: #007AFF;
  --color-primary-hover: #005ECB;
  --color-primary-alpha: rgba(0, 122, 255, 0.15);
  --radius-lg: 10px; /* Apple 标志性的大圆角 */
  --radius-md: 6px;
}

/* 深色模式映射 */
:root:not(.theme-light) {
  --color-primary: #0A84FF;
  --color-primary-hover: #409CFF;
  --color-primary-alpha: rgba(10, 132, 255, 0.2);
  
  --color-bg-app: #1C1C1E;
  --color-bg-panel: #1E1E1E;
  --color-bg-elevated: rgba(44, 44, 46, 0.85); /* 用于右键菜单的半透明质感 */
  
  --color-text-primary: #FFFFFF;
  --color-text-secondary: #8E8E93;
  
  --color-border: rgba(255, 255, 255, 0.1);
  --color-splitter-hover: var(--color-primary);
}

/* 浅色模式映射 */
:root.theme-light {
  --color-primary: #007AFF;
  --color-bg-app: #F2F2F7;
  --color-bg-panel: #FFFFFF;
  --color-bg-elevated: rgba(255, 255, 255, 0.85);
  
  --color-text-primary: #000000;
  --color-text-secondary: #8E8E93;
  
  --color-border: rgba(0, 0, 0, 0.1);
}
```

这些规则将被全面应用于 Monaco Editor 的高亮装饰、Preview 面板的节点高亮、分割条的悬停态以及我们精心打造的 ContextMenu 中。
